# Token-Onliner
Tools That Make Multiple Discord Tokens Online
![image](https://cdn.discordapp.com/attachments/892720281733431306/906162529158451220/unknown.png)
## How To Use `Token-Onliner`

### 1. Insert your tokens into the `tokens.txt` file

![image](https://cdn.discordapp.com/attachments/892720281733431306/906162709064744991/unknown.png)

### 2. Run `npm i request` & `npm i chalk` in console / shell

### 3. Run `npm i discord.js11@npm:discord.js@11.6.2`  in console / shell

### 4. Run the script with `node onliner.js` in console / shell
